About

This enhancement ports the original Python text adventure into a browser-based experience using modern HTML, CSS, and JavaScript. Game progress persists locally via your browser's storage, no external services required and fully hostable on GitHub Pages.

  • Explore rooms with directional controls
  • Collect six items of power to face Darkroot
  • Fog-of-war reveals locations as you explore
  • Progress saved automatically (localStorage)
Start Game Admin Enhancement 2 (Algorithms)

How to Play

Move with the on-screen controls (or arrow keys/WASD). If a room contains an item of power, use the Get button to add it to your inventory. When you're ready, enter Shadow Hollow. If you carry all six items, you'll defeat Darkroot. Otherwise… not so much.

Technical Notes

  • Vanilla ES Modules (no external libraries)
  • All assets are relative paths (safe for GitHub Pages)
  • Data-driven rooms and exits in ./data/rooms.json (no comments in JSON)
  • Game state stored under a namespaced key in localStorage
